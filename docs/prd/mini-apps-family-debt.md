# PRD: Mini-App - Family Debt Ledger

## Summary
Track debts and settlements between family members.

## Goals
- Record debt entries and payments.
- Show net balances per person.

## Non-Goals
- Legal accounting or interest calculations.

## User Stories
- As a user, I can record a debt entry.
- As a user, I can mark a settlement.

## Functional Requirements
- Debt entry form with debtor, creditor, amount, note.
- Settlement flow that reduces balances.
- Balance summary view.

## UX Notes
- Debts list with clear directionality (owes/owed).

## Dependencies
- Mini-apps hub route.

## Milestones
1. Debt entries and list.
2. Settlement handling.
3. Balance summary.

## Risks
- Confusion if split with shared expenses is unclear.
